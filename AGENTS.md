# Architecture Rules

## Frontend
- Atomic design
- Feature-first structure
- Tailwind + shadcn
- No business logic in UI

## Backend
- Hexagonal architecture
- CQRS where needed
- DTO validation mandatory
- Repository pattern

## AI Agents
- UX agent only modifies flows
- UI agent only modifies design system
- Code agent respects contracts package

## Commits
- Conventional commits
- Small scoped commits
